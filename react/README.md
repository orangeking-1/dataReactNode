# esLint介绍
* 官网：<http://eslint.cn/docs/user-guide/configuring>
* 代码统一风格规范有很多：jslint,jshint,eslint
## 使用
* 全局和本地安装eslint
  + `npm install eslint -g `
  + `npm install eslint -save-dev`
* 创建并编写配置文件
  + 配置文件可以写成好多种方式
  + 注释的写法
  + .eslintrc.json
  + yaml文件
  + .eslintrc.js（主流）
  + 可以配置在npm的package.json文件中
  + 可以直接配置在webpack的属性中
* 在webpack中配置eslint-loader
